# gulp-compo-builder

**Gulp** incremental builder for **[CompoJS](http://compojs.ru/)**

![logo](https://github.com/compo-js/gulp-compo-builder/blob/main/builder.png)

## Install

```
npm i gulp-compo-builder
```
